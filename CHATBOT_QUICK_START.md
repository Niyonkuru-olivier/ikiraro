# UMUHUZA Chatbot Quick Start Guide

## Files Created

1. **`chatbot_knowledge_base.json`** - Complete knowledge base with all agriculture information
2. **`CHATBOT_TRAINING_GUIDE.md`** - Detailed training instructions
3. **Updated `templates/index.html`** - Enhanced chatbot configuration

## Quick Training Steps

### Option 1: Import Knowledge Base (Recommended)
1. Log in to https://studio.botpress.cloud
2. Select your bot (ID: `f5a1ef00-3dd2-4a3e-b8b9-0ba70c5f6d37`)
3. Go to **Knowledge** or **QnA** section
4. Import `chatbot_knowledge_base.json`
5. The bot will automatically learn from this data

### Option 2: Manual Training
1. Follow the detailed instructions in `CHATBOT_TRAINING_GUIDE.md`
2. Create intents and responses as outlined
3. Test with sample questions
